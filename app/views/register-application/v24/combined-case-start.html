You’re about to apply to register a biodiversity gain site

You can also choose to record allocation of off-site gains to a development at the same time as registering the gain site.

To allocate to a development and register a gain site at the same time, you'll need:

  • all the documents needed to register a gain site
  • the completed statutory biodiversity metric tool calculations:
    • for the site you want to register
    • which the local planning authority reviewed with the development project's planning application
  • a planning application decision notice for the development project from your local planning authority

It costs £639 to allocate and register at the same time.
